# UI 정의서 — 구역별 대시보드

> 특정 구역(예: 구역 A·B)을 선택했을 때의 대시보드. **① 구역별 대시보드(전체 페이지)**와, 3D 장비를 클릭하면 열리는 **② 장비 상세** 두 화면으로 구성된다. 화면별로 정의를 분리한다.

---

## ① 구역별 대시보드 (전체 페이지)

### 화면 기본 정보

| No. | 항목명 | 내용 |
| --: | --- | --- |
| 1 | 업무 Level1 | 공장 생산 관제 |
| 2 | 업무 Level2 | 구역별 대시보드 |
| 3 | 화면 ID | `SFM-DD-001-M-0` |
| 4 | 화면명 | 구역별 대시보드 |
| 5 | 설명 | 한 구역의 생산 현황을 종합 표시한다. 오늘의 생산 달성률, 장비 요약 카드, 공정 STEP 선택, 3D 장비 플릿(대기열 포함), 하단 스케줄 간트(계획/현재 상태 탭)를 한 화면에서 제공한다. |
| 6 | 화면유형 | Main |
| 7 | 개선유형 | 신규 |
| 8 | 작성일 | 2026-06-15 |
| 9 | 작성자 | jisung |

```
┌───────────────────────────────────────────────────────────────────────────────┐

├───────────────────────────────────────────────────────────────────────────────┤
│ [가동가능 18대] [가동불가 2대] [평균가동률 72%] [총 대기 37] [평균대기 8분]      │ ← 요약 카드(5)
├───────────────────────────────────────────────────────────────────────────────┤

├───────────────────────────────────────────────────────────────────────────────┤
│  ┌ 대기열 ─┐                                                                     │

│ MACHINE-01 │   ▓▓ UNIT-117 ▓▓      ▓▓▓                                          │
│ MACHINE-02 │      ▓▓▓▓ UNIT-210                                                 │
│ MACHINE-03 │   ░░ (계획 탭=옅은 고스트) ░░                                       │
└───────────────────────────────────────────────────────────────────────────────┘
```

### 구성

상단에서 하단으로 5개 영역이 세로로 쌓인다.
1. **브레드크럼**: 전체 대시보드 ▸ 선택 구역
2. **생산 달성률 바**: 오늘 실적/목표(`daily_output_qty / daily_target_output_qty`) 기반 달성률(%)·진행바·현재/목표 수량
3. **요약 카드(5)**: 가동 가능 장비 수 · 가동 불가능 장비 수 · 평균 가동률 · 총 대기 UNIT 수 · 평균 대기 시간
4. **공정 STEP 셀렉터**: STEP 탭 전환(전환 시 3D·간트가 해당 STEP 장비로 슬라이드)
5. **3D 장비 플릿 + 하단 간트**: 3D 장비 위 정보 카드(부하율·상태·투입 UNIT), 좌측 대기열 카드(대기 UNIT 수·평균 대기), 하단 스케줄 간트(계획/현재 상태 탭 + 시뮬 현재선). 진입 시 무거운 3D는 로딩 스켈레톤 후 마운트.

### 처리절차

1. 헤더에서 구역 선택 (특정 구역)
2. 생산 달성률·요약 카드로 구역 현황 파악
3. 공정 STEP 선택 → 해당 STEP 장비/스케줄로 전환
4. 하단 간트에서 [계획]/[현재 상태] 탭 전환하여 계획 대비 실제 진행 확인 (막대 hover → 풀네임/시간 툴팁)
5. 3D 장비 클릭 → ② 장비 상세 열기

### 제약사항

- 로그인 사용자 전용
- 구역 미선택('전체')이면 전체 대시보드(구역 통합 관제)로 분기 → 본 화면 미표시
- 간트 현재선/날짜는 시뮬레이션 현재 시각 기준 (계획 탭=계획 시작일, 현재 상태 탭=실제 작업일 기준 표시)
- API 미연결·로딩/에러 시 안내 박스 표시, 표시할 데이터 없으면 빈 상태 안내

## ② 장비 상세

### 화면 기본 정보

| No. | 항목명 | 내용 |
| --: | --- | --- |
| 1 | 업무 Level1 | 공장 생산 관제 |
| 2 | 업무 Level2 | 구역별 대시보드 |
| 3 | 화면 ID | `SFM-DD-001-S-1` |
| 4 | 화면명 | 장비 상세 |
| 5 | 설명 | 3D 장비 플릿에서 장비를 클릭하면 우측에서 슬라이드되어 나오는 패널. 장비 식별/상태, 가동률·부하율, 현재 진행 중인 UNIT의 진행 상황을 제공한다. |
| 6 | 화면유형 | Sub (3D 보드에서 호출되는 슬라이드 패널) |
| 7 | 개선유형 | 신규 |
| 8 | 작성일 | 2026-06-15 |
| 9 | 작성자 | jisung |

### 화면 레이아웃

```
                                              ┌──────────────────────────────┐
                                              │ Diffusion Line A           ✕ │ ← 장비명(machine_type)+닫기
                                              │ MACHINE-01   [가동중]         │ ← 장비 ID + 상태 배지
                                              ├──────────────────────────────┤
                                              │ ┌ 가동률 ──┐  ┌ 부하율 ──┐   │
                                              │ │   72 %   │  │   80 %   │   │ ← 가동률 / 부하율 (2분할)
                                              │ └──────────┘  └──────────┘   │
                                              ├──────────────────────────────┤
                                              │ 현재 진행 중 UNIT             │
                                              │ ┌──────────────────────────┐ │
                                              │ │ UNIT-117      우선순위 2  │ │
                                              │ │ 10:00 → 13:00        64 % │ │ ← 시작→예상종료 + 진행률
                                              │ │ ▓▓▓▓▓▓▓░░░               │ │ ← 진행 바
                                              │ └──────────────────────────┘ │
                                              │  (없으면: 진행 중인 UNIT이    │
                                              │   없습니다.)                  │
                                              └──────────────────────────────┘
```

### 구성

3D 보드 우측에서 슬라이드되어 나타나는 패널로, 헤더·지표·진행 UNIT 세 영역으로 구분된다.
- **헤더**: 장비명(machine_type, 가장 큰 글씨), 장비 ID, 가동 상태 배지, 닫기(✕)
- **지표**: 가동률 / 부하율 2분할 카드
- **현재 진행 중 UNIT**: UNIT ID·우선순위 칩, 시작→예상 종료 시각, 진행률(현재 시각 기준 %)과 진행 바. 진행 중 UNIT이 없으면 빈 상태 문구.

### 처리절차

1. 3D 플릿에서 장비 클릭 → 패널 슬라이드 인 (해당 장비로 줌인)
2. 가동률·부하율 확인
3. 현재 진행 중 UNIT의 진행률·시간 확인
4. 닫기(✕) 또는 다른 장비 클릭/빈 공간 클릭으로 닫기

### 제약사항

- 3D 장비 선택 시에만 표시
- 진행률은 시뮬레이션/현재 시각과 UNIT 시작·예상 종료로 계산 (0~100% 클램프)
- 진행 중(status='진행중') UNIT이 없으면 빈 상태 표시
- API 미연결 시 데모(mock) 데이터로 표시
