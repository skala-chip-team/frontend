전체 총괄자

기능 설명

전체 총괄자는 작업자 계정 목록을 조회하고, 각 작업자의 역할과 권한 구역을 변경할 수 있다.

시스템은 작업자별로 역할과 담당 구역, 계정 상태를 연결하여 관리하며, 이를 기반으로 화면 접근 권한과 구역별 운영 책임을 통제할 수 있도록 지원해야 한다.

역할은 운영자와 작업자 중에서 지정하며 관리자 계정의 역할은 변경할 수 없다. 한 작업자에게 하나 이상의 권한 구역을 배정할 수 있어야 한다.

또한 역할과 권한 구역 변경 이력을 관리하여 운영 권한 변경 사항을 추적할 수 있어야 한다.
