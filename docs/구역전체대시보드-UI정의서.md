# UI 정의서 — 구역 전체 대시보드

> 전체 대시보드(구역 통합 관제)는 3D 공장 뷰를 중심으로, 사용자의 상호작용에 따라 세 가지 상태 화면으로 전개된다.
> **① 구역을 클릭했을 때**(구역 상세) / **② 장비를 클릭했을 때**(장비 상세) / **③ 유닛 경로 보기를 클릭했을 때**(공정 경로 강조).
> 세 화면 모두 동일한 3D 캔버스 위에 오버레이/줌으로 동작하며, 화면별로 정의를 분리한다.

### 구성

3D 공장 뷰를 배경으로, 상단의 구역 KPI 바와 우측의 정보 컬럼(2단)으로 구성된 오버레이 화면이다. 좌상단에는 데이터 연결 상태 배지(불러오는 중 / API 연결 실패·데모 / 데모 데이터)가 표시된다. 상단 KPI 바에는 전체로 돌아가는 버튼, 구역 라벨, 그리고 평균 가동률·가용/전체 장비 수(다운 대수 ▼ 강조)·대기 유닛·평균 대기·일일 생산·재조정 건수 KPI가 나열된다. 우측 컬럼 상단에는 해당 구역의 최신 재조정안 카드(위험 등급 칩, 원인, 발생 시각·최대 위험도·영향 UNIT, 재조정하기 버튼)가, 하단에는 step별 대기 큐 카드가 배치된다.

### 처리절차

1. 전체 3D 뷰에서 공정 구역 클릭 → 해당 구역으로 줌인
2. 구역 KPI 확인 (가동률·가용 장비·대기·재조정 등)
3. 최신 재조정안 확인 → 필요 시 [재조정하기]로 재조정 상세 이동
4. 대기 큐(step별) 확인
5. 장비 클릭 → ② 장비 상세 / 빈 공간 클릭 → 전체로 복귀

### 제약사항

- 로그인 사용자 전용
- 구역 미선택(전체) 상태에서는 "공정 구역을 클릭하면 확대되어 상세 정보가 표시됩니다." 안내만 노출
- 장애 지속·일일 생산 등 일부 지표는 시뮬레이션 현재 시각 기준
- API 미연결 시 데모(mock) 데이터로 표시

### 화면 레이아웃

```
                                                   ┌───────────────────────────┐
                                                   │ 장비 상세               ✕ │ ← 닫기(구역 상세로)
                                                   │ MACHINE-12 [STEP B][위험] │   위험=장애/정지 시
                                                   │                  ● 장애    │   상태 점+색
                                                   │ ┌ 장애 지속 ────────────┐ │ ← 장애/정지 시에만
                                                   │ │ 3.5시간째   감지 11:20 │ │
                                                   │ └───────────────────────┘ │
                                                   │ ┌───────────────────────┐ │
                                                   │ │  ◠72%◡   가동률  68%   │ │ ← 부하율 원형차트(메인)
                                                   │ │  부하율  투입UNIT UN-..│ │
                                                   │ │          소속구역 구역A│ │
                                                   │ └───────────────────────┘ │
                                                   │ [ UNIT-.. 공정 순서 보기 ] │ ← 투입 UNIT 있을 때 → ③
                                                   └───────────────────────────┘
```

### 구성

구역 상세 화면의 우측 상단 카드가 "장비 상세"로 전환된 형태다. 헤더에는 장비 ID, STEP 칩, 위험 장비 칩(장애·정지 시), 가동 상태(점+색), 닫기 버튼이 배치된다. 장애·정지 상태일 때는 장애 지속 시간과 감지 시각을 빨강 박스로 강조한다. 본문 중앙에는 부하율 원형 차트(메인 지표)와 함께 가동률·투입 UNIT·소속 구역을 보여준다. 투입 UNIT이 있으면 하단에 "공정 순서 보기" 버튼(→ ③)이 표시된다. 이때 우측 하단의 대기 큐 카드는 숨겨진다.

### 처리절차

1. 구역 줌인 상태에서 3D 장비 클릭 → 우측 카드가 장비 상세로 전환
2. 부하율·가동률·투입 UNIT·상태 확인
3. (장애 시) 장애 지속 시간·감지 시각 확인
4. [공정 순서 보기] → ③ 유닛 경로 보기 / 닫기(✕) → 구역 상세로

### 제약사항

- 구역 줌인 상태에서 장비 선택 시에만 표시
- 장애 지속 시간 = 정지 시작(fault_since) ↔ 시뮬레이션 현재 시각 차이 (시뮬 정지 시 "—")
- "공정 순서 보기"는 투입 UNIT이 있을 때만 노출
- API 미연결 시 데모(mock) 데이터로 표시

## ③ 유닛 경로 보기를 클릭했을 때 — 공정 경로 강조

### 화면 기본 정보

| No. | 항목명 | 내용 |

| 6 | 화면유형 | Sub (3D 경로 강조 상태) |

### 화면 레이아웃

```
┌──────────────────────────────────────────────────────────────────────────────┐
│ ┌────────────────────────────────────────────────────────────────────────────┐│
│ │ [← 전체]  구역 A   …(구역 KPI 바 유지)…                                      ││
│ └────────────────────────────────────────────────────────────────────────────┘│
│                                                   ┌───────────────────────────┐│
│    [ 3D 공장 — 선택 UNIT 경로 강조 ]              │ 장비 상세               ✕ ││
│                                                   │ MACHINE-12 …               ││
│   STEP A ─▶ B ─▶ C ─▶ D                           │ …(부하율/가동률 카드 유지) ││
│   ●━━━━━━●━━━━━━●━━━━━━●   ← 담당 장비 경로 밝게   │                            ││
│   (그 외 장비/구역은 흐리게 dim 처리)             │ [ UNIT-.. 공정 순서 숨기기]││ ← 버튼 라벨 토글
│                                                   └───────────────────────────┘│
└──────────────────────────────────────────────────────────────────────────────┘
```

### 구성

장비 상세 화면을 유지한 채, 3D 공장 뷰만 "경로 강조 모드"로 전환된 상태다. 선택된 UNIT(투입 UNIT)이 거치는 공정 순서(STEP A→B→C→D)의 담당 장비들을 따라 경로가 밝게 표시되고, 경로에 속하지 않는 장비·구역은 흐리게(dim) 처리되어 동선이 부각된다. 우측 장비 상세 카드의 버튼 라벨은 "공정 순서 보기"에서 "공정 순서 숨기기"로 바뀌어 토글 상태를 나타낸다.

### 처리절차

1. 장비 상세에서 [UNIT-.. 공정 순서 보기] 클릭 → 3D 경로 강조 모드 진입
2. 해당 UNIT의 STEP A→B→C→D 담당 장비 경로 확인
3. [공정 순서 숨기기] 재클릭 → 경로 강조 해제(일반 뷰 복귀)
4. 다른 장비 클릭/장비 닫기/빈 공간 클릭 시 경로 강조 자동 해제

### 제약사항

- 장비 상세에 투입 UNIT이 있을 때만 진입 가능
- 동시에 하나의 UNIT 경로만 강조
- 장비 선택 변경·구역 이탈 시 강조 상태 초기화
- API 미연결 시 데모(mock) 데이터로 표시
