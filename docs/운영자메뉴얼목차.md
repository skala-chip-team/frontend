7.2 주요 장애 시나리오별 대응 절차

7.2.1 백엔드 파드 전체 다운

증상 확인 → 로그 조회 → 재기동 → 헬스체크

7.2.2 PostgreSQL 접속 불가

커넥션 풀 포화 확인 → DB 재시작 → 백업 복구 분기

7.2.3 XGBoost 모델 추론 실패

fallback 동작 확인 → 모델 파일 무결성 검사 → 이전 버전 롤백

7.2.4 LangGraph 에이전트 응답 없음

개별 에이전트 로그 확인 → Supervisor 재기동 → 에이전트 순서 재기동

7.2.5 재조정 스케줄러 미실행

크론 상태 확인 → 수동 트리거 실행 → pending 그룹 상태 점검
